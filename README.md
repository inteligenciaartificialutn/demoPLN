# demoPLN
Ejemplos de Procesamiento del Lenguaje Natural.

## Instalación
1. Acceder a [Google Drive](https://drive.google.com/drive/my-drive) usando una cuenta disponible.
2. Crear la carpeta "**demosColab**" en el raíz de la unidad (si no existe). 
3. Crear la carpeta "**demoPLN**" dentro de la carpeta "demosColab" (ya existente o creada en el paso anterior).
4. Subir todos los archivos de este repositorio dentro de la carpeta creada en el paso anterior. 

## Uso
Para correr los ejemplos:
1. Abrir con la aplicación <**Google Colaboratory**> haciendo click derecho en el cuaderno **.ipynb** seleccionado.
2. Una vez abierto el entorno, si se considera necesario, ajustar los parámetros correspondientes del ejemplo.
3. Ejecutar todas las celdas usando la opción <*Ejecutar todas*>.
